# 💰 Mini Project 3: Quiz Keuangan (TVM Edition)

Proyek ketiga dalam perjalanan belajar Frontend Development saya. Aplikasi kuis interaktif yang membahas konsep **Time Value of Money (TVM)** menggunakan Vanilla JavaScript.

## 🚀 Fitur Utama
- **Dynamic Question Rendering**: Soal dimuat secara dinamis dari Array of Objects.
- **Auto-Progress**: Soal otomatis berpindah saat jawaban dipilih menggunakan `change` event.
- **Smart Scoring**: Penilaian akhir berdasarkan kategori skor dengan pesan yang dipersonalisasi.
- **Responsive UI**: Desain ceria dan responsif menggunakan CSS Flexbox dan variabel `:root`.

## 🛠️ Tech Stack
- **HTML5**: Struktur semantik.
- **CSS3**: Custom styling dengan efek shadow dan hover.
- **JavaScript (ES6)**: Manipulasi DOM, Looping, dan logika kuis.

## 🧠 Pelajaran Baru (Key Learnings)
Dalam proyek ini, saya berhasil memecahkan tantangan:
1. Menggunakan **Looping** untuk memunculkan pilihan jawaban secara otomatis.
2. Mempelajari **Event Listener 'change'** pada input radio.
3. Melakukan **DOM Cleansing** (`innerHTML = ""`) sebelum merender data baru agar aplikasi tidak tumpang tindih.
