# Simple Responsive Navbar 
Ini adalah mini proyek latihan pertama saya dalam membangun komponen web yang interaktif. Fokus utama proyek mini ini adalah mengimplementasikan manipulasi DOM menggunakan JavaScript sederhana untuk menciptakan menu yang responsif.

## Fitur
- **Responsive Design** :
    Tampilan menyesuaikan antara layar desktop dan smartphone.
- **Interactive Toggle** :
    Menu Dropdown yang berfungsi saat tombol menu di klik pada versi mobile.
- **Clean Code** :
    Ditulis dengan HTML5, CSS3, dan Vanilla JavaScript tanpa library eksternal.

## Teknologi Yang Digunakan
- **HTML5** :
    Untuk struktur navigasi.
- **CSS3** :
    Menggunakan Flexbox untuk tata letak dan Media Queries untuk responsivitas.
- **JavaScript (ES6)** :
    Menggunakan `addEventListener` dan `classList.toggle` untuk logika menu.
- **Code Editor** :
    VS Code.
- **Operating System** :
    Linux Mint XFCE.
- **Laptop** :
    Lenovo G40-70.

## Pelajaran Yang Diambil
Melalui proyek ini, saya memperdalam pemahaman tentang :
1. Bagaimana menghubungkan file JavaScript ke HTML.
2. Cara kerja event listener `click` pada elemen tombol.
3. Mengontrol tampilan elemen CSS melalui JavaScript.
