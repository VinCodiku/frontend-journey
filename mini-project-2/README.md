# Simple To-Do List

Ini adalah mini proyek latihan kedua saya dalam membangun komponen web yang interaktif. Fokus utama proyek ini adalah masih dengan mengimplementasikan JavaScript sederhana untuk membuat To-Do List sederhana.

### Fitur 
- Tambah tugas lewat form (dukungan tombol Enter).
- Hapus tugas secara spesifik.
- Desain cantik (Earth Tone Theme).

## Teknologi Yang Digunakan 
- HTML5
- CSS3
- JavaScript
- VS Code (Code Editor)    
- Linux Mint XFCE (OS)
- Lenovo G40-70 (Laptop)

## Pelajaran Yang Diambil 
1. DOM Manipulation (`createElement`, `appendChild`, `removeChild`).
2. Event Listeners (`submit`, `click`).
3. Dasar CSS Flexbox.

*Dibuat sebagai bagian dari perjalanan belajar JavaScript saya.*
